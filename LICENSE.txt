OBS: PERGUNTAS FREQUENTES

1. O que é Estoque de Modelos?

O Template Stock é uma iniciativa da AgileITs para fornecer designs da Web gratuitos que são compatíveis com vários dispositivos.

2. Os modelos de estoque de modelos são realmente gratuitos?

Sim, todos os nossos modelos são gratuitos para uso comercial e não comercial, mas você fornece um link para o Template Stock.com, que já está incluído no design de rodapé do Template Stock.com, não o edite ou remova.

3. Quero Ajudar Estoque de Modelos, Como posso?

Você pode ajudar Template Stock By
1. Doe alguns $, qualquer valor que desejar
2. Contribuir com o inventário de design, como fotos, ícones ou designs PSD com todos os direitos para o Template Stock

4. Quero remover o link de retorno do Template Stock.com do rodapé?

Temos dois planos para isso por modelo e ilimitado.
Doe-nos $ 10 por modelo. Se você quiser modelos para vários domínios ou modelos em massa, entre em contato com info@templatestock.co

5. Esses modelos funcionarão no iPhone, plataformas Android, Tabs como Kindle e Ipads?

Sim, os modelos do Template Stock funcionam com todos os smartphones e tablets. Para suportar todos os dispositivos que estamos fornecendo bootstrap Responisve designs WEB Template.

6. O que é Modelo Web?

O modelo WEB é um design responsivo que pode ser usado para usuários de desktop. Os usuários que visitam o site a partir de navegadores de desktop podem visualizar o modelo WEB


7. Preciso de uma versão separada para Smartphones e Tablets?

Não, o WEB Template é compatível com todos os navegadores web, Smartphones e Tablets.

8. Preciso de algum banco de dados?

Não, não é necessário.


11. Sob qual licença você está fornecendo esses modelos?
